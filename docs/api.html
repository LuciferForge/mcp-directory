Quick start

After subscribing, your license key arrives via email. Use it as X-API-Key:

# list 10 highest-volume crypto markets
curl "https://api.protodex.io/markets?category=crypto&limit=10" \
  -H "X-API-Key: pdx_YOUR_KEY"

# pull 1-hour price candles for a market over the last 30 days
curl "https://api.protodex.io/markets/12345/prices?interval=1h" \
  -H "X-API-Key: pdx_YOUR_KEY"

# subscribe to live crash signals over WebSocket
wscat -c "wss://api.protodex.io/ws/crashes?api_key=pdx_YOUR_KEY"

Free tier (no key) gives you 100 req/day per IP, perfect for trying the API. Subscribers get 100,000 req/day, no rate-limit headers needed.
